Visualização da primeira entrega do Projeto

Objetivo

  • Preparar o projeto para ser testado

 

Visualização

O objetivo da visualização é determinar que funcionalidades do projeto foram implementadas com sucesso.
A visualização é realizada através da execução de testes de integração preparados pelos docentes.

Pede-se a cada grupo que esta semana compareça no laboratório para preparar os servidores para serem testados.
Para tal devem:

  1. Obter o código entregue
  2. Seguir as instruções contidas no ficheiro README.md elaborado pelo próprio grupo
  3. Compilar e testar todo o sistema, para confirmar que está operacional

Quando os servidores estiverem a funcionar, devem chamar o docente do laboratório para que este execute os testes de integração e registe os resultados obtidos.
O docente irá também observar os testes preparados pelo grupo e pedir alguns esclarecimentos sobre o trabalho desenvolvido.

Enquanto esperam ou depois de mostrar o projeto, devem aproveitar para experimentar os exemplos de Criptografia.
