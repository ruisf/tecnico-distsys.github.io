Apoio ao Projeto

Objetivos de Projeto

  • Completar as funcionalidades dos serviços
  • Concluir testes de integração do broker-ws

Resumo das tarefas do projeto

  • Substituir cxx nas coordenadas dos módulos Maven pelo código do grupo:
    substituir c por a (Alameda) ou t (Taguspark) e substituir xx por número do grupo com dois dígitos.
    Por exemplo, o grupo 99 do Taguspark deve substituir por t99.
  • Editar README.md com a identificação do grupo
  • Transportadoras
    1. Construir "esqueleto" do servidor
      • Implementar operação ping()
    2. Construir "esqueleto" do cliente
      • Invocar a operação ping()
    3. Implementar operações do serviço
      • E respetivos testes unitários
    4. Concretizar registo e pesquisa no UDDI
      • E respetivos testes unitários usando mocks para simular UDDI
    5. Testes de integração das operações principais
      • Testar casos mais importantes
      • Não esquecendo casos com entradas incorretas: null, "", valores inesperados, etc.
    6. Lançar e registar múltiplas instâncias de transportadoras: UpaTransporter1, UpaTransporter2, ...
    7. Atualizar README.md com instruções
  • Corretor
    1. Construir "esqueleto" do servidor
      • Implementar operação ping()
    2. Construir "esqueleto" do cliente
      • Invocar a operação ping()
    3. Tornar o corretor cliente de transportadoras
      • Sugestão: acrescentar dependência Maven para módulo transporter-ws-cli
      • Sugestão: fazer a operação ping() do corretor invocar o ping() de cada transportadora
    4. Concretizar registo e pesquisa no UDDI
      • E respetivos testes unitários usando mocks para simular UDDI
    5. Implementar operações do serviço
      • E respetivos testes unitários usando mocks para simular Transporters, se necessário
    6. Testes de integração das operações principais
      • Testar casos mais importantes - abrangem toda a solução em funcionamento
      • Mais uma vez, não esquecer os casos com entradas incorretas: null, "", valores inesperados, etc.
    7. Finalizar instruções no README.md
